# GDUT 校内通知定时获取器

（准备）服务于小程序，静态化常用（通知）数据，结合Flask提供后台Restful数据服务

## 环境要求

* Python 2.7
* Windows 7

## 项目进展
* 2017-05-15 完成定时获取通知标题并静态化成JSON文件

## 使用方法
* 启用定时器

```
$ python infoGetter.py
```
